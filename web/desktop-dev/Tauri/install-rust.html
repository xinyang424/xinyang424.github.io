          运行

          1. 下载后的文件叫rustup-init.exe，双击运行，到下图输入1后回车：
          2. 等待下载完成，会提示让你回车以继续press enter to continue
          3. 安装目录在C:\Users\admin\.cargo\bin
          4. 将上面的安装目录添加进Path环境变量
          5. 终端输入rustup回车后不报错即安装成功
          6. rust使用参考文档open in new window

          常用命令

          1. 更新
          rustup update
          1. 卸载
          rustup self uninstall
          1. 查看版本
          rustc --version
